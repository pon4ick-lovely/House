▎README

▎Цель проекта

Данный проект направлен на изучение работы базовых магических методов в Python на практике. Мы создадим класс House, который будет демонстрировать использование магических методов __len__ и __str__.

▎Задача "Магические здания"

В этой задаче мы будем расширять класс House, добавляя специальные методы для получения информации о здании.

▎Задание

Необходимо дополнить класс House следующими магическими методами:

• __len__(self): должен возвращать количество этажей здания, хранящееся в атрибуте self.number_of_floors.

• __str__(self): должен возвращать строку в формате: "Название: <название>, кол-во этажей: <этажи>".

▎Пример реализации

class House:
    def __init__(self, name, number_of_floors):
        self.name = name
        self.number_of_floors = number_of_floors

    def __len__(self):
        return self.number_of_floors

    def __str__(self):
        return f"Название: {self.name}, кол-во этажей: {self.number_of_floors}"

# Пример выполняемого кода:
h1 = House('ЖК Эльбрус', 10)
h2 = House('ЖК Акация', 20)

# str
print(h1)  # Вывод: Название: ЖК Эльбрус, кол-во этажей: 10
print(h2)  # Вывод: Название: ЖК Акация, кол-во этажей: 20

# len
print(len(h1))  # Вывод: 10
print(len(h2))  # Вывод: 20


▎Ожидаемый вывод

При выполнении приведенного выше кода вы получите следующий вывод в консоль:

Название: ЖК Эльбрус, кол-во этажей: 10
Название: ЖК Акация, кол-во этажей: 20
10
20
